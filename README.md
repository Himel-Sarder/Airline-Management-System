## Installation 🛠️

1. **Clone the Repository** 📥:
 ```bash
 git clone https://github.com/username/Airline-Management-System.git
 cd Airline-Management-System
 ```

2. **Install Dependencies** 📦:
 ```bash
 pip install -r requirements.txt
 ```

## Usage 🎯

1. **Run the Application** 🚀:
 ```bash
 python app2.py
 ```
